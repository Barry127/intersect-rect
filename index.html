Intersect-rect

Intersect-rect is a tiny module to check if two DOM elements intersect (overlap). See the example below.

Example

Drag the boxes below:

Result: A & B do not intersect
A B 

/* Drag code is simplified */

function overlap() {
  var A = document.getElementById('#a');
  var B = document.getElementById('#b');

  var rectA = A.getBoundingClientRect();
  var rectB = B.getBoundingClientRect();

  if (intersectRect(A, B)) {
    // Set result to green
  } else {
    // Set result to red
  }
}

$('#a').on('drag', overlap);
$('#b').on('drag', overlap);

Install

Intersect rect can be installed using npm:

npm install --save intersect-rect

Loading intersect-rect using common JS:

var intersectRect = require('intersect-rect');

The module alose works out of the box in the browser or with AMD modules.
